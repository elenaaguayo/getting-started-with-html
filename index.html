THE BASICS OF THE WEB AND HTML - STAGE 0

The Web
Invented in the 1990's, now has 30 billion pages.
World Wide Web (WWW) is a collection of HTML (Hypertext Markup Language) documents.
Major pieces of web = browser, internet and servers.

HTML
Made up of text, content, markup, references, links...
Markup is made og tags like < b > < / b > for bold and < em > < / em > for italics

HTML ATTRIBUTES
ATTR are values, they can have various attributes
Anchor tag to make a link
< a > < / a > , < href = "web to go" > derp < / a >
Image tag to insert an image
< img src = "url" alt = "text" >
White space
< br > is break, inline, ends a line
< p > < / p > for paragraph, block, makes like an invisible box around the text

HTML documents
Doctype, element, head (meta-data, including tittle), body tag (includes content of document,
what we see in the screen) and finally a < / html > tag.
